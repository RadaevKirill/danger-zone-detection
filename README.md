# danger-zone-detection
Обнаружение людей находящихся в запретной зоне

## Install
```shell
python3 -m venv venv
source venv/bin/activate

pip install -U pip
pip install -r requirements.txt

# Обязательно устанавливается отдельно
pip install nptyping
```

## Usage
```shell
# Запуск основого приложения
python3 main.py

# Запуск конфигуратора зоны
python3 zone_configurator.py
```

## Hints for main app
Конфигурирование приложения осуществляется с помощью файла `assets/config.yaml`,
в нем можено поменять попрог уверенности модель и т.д

В проект предварительно подгружено две модели YOLO V11n и V11s находящиеся по пути 
`assets/yolo11*.pt`

## Hints for config areas
При запуске приложения вы увидите первый кадр из входного видео,
для данного приложения не было сделано возможности гибкой конфигурации, при работе
обращайте на это внимание.

При работе с ним кликайте по точкам, которые в будующем будут являться вершинами полигона который будет анализироваться
После того как выделили все точки, нажмите на любую клавишу (кроме 'q'), для показа выбранной области.
Что бы сохранить область нажмите 'q'
